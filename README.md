# 🌀 VelocityOptic

A dedicated motion blur mod for **Minecraft 1.21.11** built on the Fabric Loader.

## 🚀 Features
- **Dynamic Sampling:** Calculates blur based on movement delta.
- **Ultra-Lightweight:** Zero impact on tick-rate, minimal GPU overhead.
- **Native 1.21.11 Support:** Tailored for the latest rendering pipeline.

## 📂 Structure
- `src/main/java`: Mod source code.
- `src/main/resources`: Shader assets and mod metadata.

## 🛠 Setup
1. Clone the repo.
2. Run `./gradlew build`.
3. Locate the jar in `build/libs` and drop it into your 1.21.11 mods folder.
